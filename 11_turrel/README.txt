# Восприятие урона

  
В данный момент у башни есть только один “**орган чувств**” — зрение. Это позволяет башне замечать игрока, когда он находится в поле зрения, но тем не менее она полностью будет его игнорировать, если он решит открыть огонь.

Для этого необходимо добавить новое чувство в **Perception component** контроллера башни - **AISense_Damage** и в тот момент, когда игрок будет наносить урон по башне, то башня должна быстро повернуться в его сторону.

## Рекомендации по выполнению

Для того, чтобы сообщить **AI Controller**’у у эктора информацию об уроне, недостаточно просто вызвать метод **TakeDamage**. Для этого необходимо воспользоваться статическим методом **ReportDamageEvent** у класса **UAISense_Damage**

```cpp
ReportDamageEvent(UObject* WorldContextObject, AActor* DamagedActor, AActor* Instigator, float DamageAmount, FVector EventLocation, FVector HitLocation)
```

Здесь в качестве **Instigator**’а рекомендуется передавать **Actor**’а, который нанес урон, а в качестве **EventLocation** — либо местоположение игрока, либо начало выстрела.

## Дополнительно

Реализовать здоровье башни. Можно в виде атрибута самого эктора, можно в виде отдельной компоненты. При достижении 0, башня взрывается (можно использовать эффект взрыва гранаты, при этом урон **НЕ** наносится), ее внешнее представление исчезает, а логика перестает работать.
